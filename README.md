# DMC Keeper

Short of it: An inventory application for DMC thread.

Long of it: Single-module Spring Boot project scaffolded from Spring Initializr and built with Gradle (Boot 4.1 on Java 21), wired for Postgres through Spring Data JPA.

## Tech stack

### Backend

- Java 21 
- Spring Boot 4.1 / Spring MVC (REST) + Spring Data JPA/Hibernate
- Gradle
- Testing with JUnit 5

### Database

- Postgres (run locally via Docker Compose)

### Frontend (planned)

- React/TypeScript
- Vite
